ШАБЛОН ФИНАНСОВОЙ МОДЕЛИ ДЛЯ ФРАНЧАЙЗИ
1.	Назначение документа
1.1. Настоящий шаблон описывает структуру финансовой модели, которая предоставляется потенциальному франчайзи для оценки запуска центра.
2.	Основные блоки модели
2.1. Инвестиции в открытие:
•	ремонт и адаптация;
•	мебель и оборудование;
•	стартовый маркетинг;
•	фонд оплаты труда на период запуска;
•	финансовая подушка.
2.2. Доходная часть:
•	средний чек;
•	количество клиентов / загрузка;
•	дополнительные услуги: [[ЗАПОЛНИТЬ]].
2.3. Расходная часть:
•	ФОТ;
•	аренда / содержание объекта;
•	питание;
•	коммунальные расходы;
•	маркетинг;
•	роялти;
•	прочие расходы.
2.4. Аналитика:
•	точка безубыточности;
•	валовая и чистая прибыль;
•	срок окупаемости;
•	сценарии: консервативный / базовый / агрессивный.
3.	Переменные для настройки
3.1. Город / регион: [[ЗАПОЛНИТЬ]].
3.2. Вместимость центра: [[ЗАПОЛНИТЬ]].
3.3. Стартовая и целевая загрузка: [[ЗАПОЛНИТЬ]].
3.4. Ставка аренды / стоимость содержания: [[ЗАПОЛНИТЬ]].
3.5. Прочие локальные параметры: [[ЗАПОЛНИТЬ]].
4.	Дисклеймер
4.1. Финансовая модель носит расчетный характер и не является гарантией достижения конкретных результатов.
